# Preferences

最終更新: 2026-05-05

## 出力フォーマット

- 日本語で回答する
- 構造化された出力（箇条書き・表・見出し）を好む
- 抽象論より具体的なコマンド・ファイルパス・コード例を好む
- 長い説明より簡潔な要点 + 詳細リンクの形式を好む
- タスクには緊急度（🔴🟡🟢）・優先度（⭐）・工数（S/M/L）を付与する形式を好む

## ツール・環境

- エディタ: Obsidian（メインのナレッジベース）
- AI CLI: Kiro CLI（メイン）、Claude Code
- OS: macOS
- バージョン管理: Git / GitLab

## 作業スタイル

- 情報は `60_Resources/`（生データ）→ `30_Notes/`（ナレッジ）→ `40_Tasks/`（タスク）の層で管理する
- 新規ファイル作成より既存ファイルへの追記を優先する
- タスクは詳細ファイル（`40_Tasks/Tasks/`）に分解して管理する
- daily_analysis → knowledge_builder の2段階フローで情報を処理する

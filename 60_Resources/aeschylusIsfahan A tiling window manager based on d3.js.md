### 概要
Isfahanは、d3.jsをベースにしたJavaScriptライブラリで、Webページ上でタイリング形式のウィンドウ環境を作成するためのツールです。

### 主な特徴と情報
- **用途**: Webアプリケーション内でのウィンドウ管理（タイリング環境の構築）
- **採用実績**: [Mirador](https://github.com/IIIF/mirador) や [mekarpeles/browser](https://github.com/mekarpeles/browser) などで使用されています。
- **ステータス**: 最終更新は2015年頃。スター数5、フォーク数3。

### 導入手順
1. **Node.jsのインストール**: 公式サイトからインストール。
2. **クローン**: `git clone https://github.com/aeschylus/Isfahan.git`
3. **依存関係の解決**: `npm install`
4. **ツール設定**: `gulp` をグローバルインストール。

### 開発の始め方
- `examples` ディレクトリへ移動。
- `run.sh`（Pythonが必要）を実行してビルドとサーバー起動を行う。
- ブラウザで `localhost:8000` にアクセス。

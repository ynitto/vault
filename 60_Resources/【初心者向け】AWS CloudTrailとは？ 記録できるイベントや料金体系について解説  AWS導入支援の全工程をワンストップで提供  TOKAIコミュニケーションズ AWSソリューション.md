### AWS CloudTrailの要約
AWS CloudTrailは、AWSアカウント内での操作（API呼び出しやマネジメントコンソール操作）を記録し、内部統制やセキュリティ強化を実現する証跡管理サービスです。

### 主な要点
* **記録できるイベントの種類**
    * **管理イベント**: リソースの設定変更やログイン操作など。
    * **データイベント**: S3バケット操作やLambda関数実行など。
    * **Insightsイベント**: API呼び出しの異常な挙動を分析。
* **導入のメリット**
    * セキュリティ強化（不正アクセスの検知・分析）
    * 迅速なトラブルシューティング（原因の早期特定）
    * コンプライアンス遵守および内部統制の強化
* **料金体系**
    * 直近90日間の管理イベントは無料。
    * それ以降や、データイベント、Insightsイベントの分析にはデータ量に応じた料金が発生。
* **活用方法**
    * インシデント発生時のユーザー追跡や、規制基準（PCI DSS等）への対応、Amazon CloudWatchとの連携によるログ監視・アラート設定に有効。

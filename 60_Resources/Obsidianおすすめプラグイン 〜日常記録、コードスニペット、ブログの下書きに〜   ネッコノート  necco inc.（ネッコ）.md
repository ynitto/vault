### 記事の概要
Obsidianを日常記録や開発・ブログ執筆の効率化に活用する佐藤あゆみ氏による、おすすめのプラグイン紹介記事です。複雑な設定を避け、テキストエディタとして拡張性を活かす運用術が解説されています。

### 目的別おすすめプラグイン

#### 1. 日々の記録
- **Thino**: X（旧Twitter）風のタイムライン形式で手軽にメモを残せる。
- **Dataview**: メモをデータベースのように検索・一覧表示できる。
- **Tag Wrangler**: タグのリネームなど管理・操作を強化。
- **Recent Files for Obsidian**: 直近で開いたファイルへ即座にアクセス。

#### 2. コード・技術管理
- **Obsidian Shiki Plugin**: コードブロックのシンタックスハイライトを実現。
- **Obsidian Embedded Code Title Plugin**: コードスニペットにファイル名のタイトルを表示。

#### 3. ブログ執筆・作業効率化
- **Templater**: テンプレート機能によりメモ作成を自動化・効率化。
- **Obsidian Auto Link Title**: URLから自動でページタイトルを取得してマークダウン化。
- **Obsidian Hover Editor**: メモをポップアップで開いて編集。

#### 4. その他・環境整備
- **Local REST API**: 外部ツール連携（公式Clipper併用時など）。
- **Obsidian Emo Uploader**: 画像ファイルを外部サーバーへ自動アップロード。
- **Hide Folders**: Obsidianで不要なフォルダを非表示化。

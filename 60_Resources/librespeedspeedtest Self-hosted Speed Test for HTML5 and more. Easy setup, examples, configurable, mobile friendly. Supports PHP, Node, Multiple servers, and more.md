### LibreSpeed プロジェクト概要
LibreSpeed は、Flash、Java、Websocket を一切使用しない、非常に軽量な HTML5 ベースのインターネット速度測定ツールです。

### 主な特徴
- **測定項目**: ダウンロード、アップロード、Ping、ジッター。
- **付加機能**: IP/ISP表示、テレメトリ、結果共有、複数サーバー対応。
- **互換性**: IE11 を含む全現代的なブラウザおよびモバイルに対応。

### サーバー要件
- **Webサーバー**: Apache, Nginx, IIS 等。
- **環境**: PHP 5.4 以上。
- **DB**: MariaDB, MySQL, MSSQL, PostgreSQL, SQLite に対応。

### 開発・運用情報
- **導入**: ソースをダウンロードし、Webサーバーのルートに展開するだけのシンプルな構成。
- **デザイン**: 従来のクラシック版と最新のモダン版を `config.json` や URL パラメータで切替可能。
- **配布**: Docker コンテナイメージが提供されており、毎週更新されています。

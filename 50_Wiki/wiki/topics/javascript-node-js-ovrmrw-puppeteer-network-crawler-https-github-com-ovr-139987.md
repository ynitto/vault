# ovrmrw/puppeteer-network-crawler

## 概要

`puppeteer-network-crawler`は、GoogleのPuppeteerを活用して、Webサイトのネットワークリクエストの発生タイミングを詳細に取得するためのライブラリです。

## 詳細

- `puppeteer-network-crawler`は、GoogleのPuppeteerを活用して、Webサイトのネットワークリクエストの発生タイミングを詳細に取得するためのライブラリです。
- 主な特徴
- **目的**: Webページのネットワーク通信を監視・記録する。
- **実装**: TypeScriptベースで構築されており、Node.js環境での利用を想定。
- **導入**: npm経由でのインストールと実行が容易。
- セットアップと実行
- **インストール**: `npm install puppeteer-network-crawler`
- **開発環境**: `npm install`で依存関係を解決。
- **実行**: `npm start`でサンプルコードを実行可能。

# Web visual editors

## 概要

ブラウザ上で複数ペインやノードを視覚的に操作するための UI ライブラリ／エディタ群。アプリ内ワークスペースやフロー構築 UI の参考実装になる。

## 主要なトピック

- [[javascript]]

## 詳細

- `Isfahan` は d3.js を土台に、Web ページ上でタイル型ウィンドウ管理を行う
- `NEditorJS` はプロセスフローや自動化 UI を作るための実験的ノードエディタで、ビジュアルワークフロー構築の参考になる

# Agent Skillsを行動評価する

## 概要

AIエージェントの「Agent Skills」において、不安定なアウトプット（成果物）を直接評価するのではなく、**人事評価における「行動評価（コンピテンシー評価）」の概念を導入し、定義された手順（SKILL.md）通りにプロセスが遂行されているかを評価する手法**の提案。

## 詳細

- AIエージェントの「Agent Skills」において、不安定なアウトプット（成果物）を直接評価するのではなく、**人事評価における「行動評価（コンピテンシー評価）」の概念を導入し、定義された手順（SKILL.md）通りにプロセスが遂行されているかを評価する手法**の提案。
- 要点
- **評価の課題**
- AIのアウトプットは毎回変化し、絶対的な正解を定義するのが困難。
- アウトプットのみを基準にすると、属人化や品質のバラつきが生じやすい。
- **解決策：行動評価への転換**
- 「何を作ったか（結果）」ではなく「SKILL.mdの手順・ルールに従って処理したか（プロセス）」を評価軸にする。
- **具体的な評価フロー**
- 1. **セッションの特定**: Claude Code等のログを解析し、スキルが実行されたセッションを抽出する。

# クォータ

## 概要

本ドキュメントは、Amazon CloudFrontにおける各種制限事項（クォータ）の一覧です。多くの項目はService QuotasやAWSサポートを通じて引き上げが可能です。

## 詳細

- 本ドキュメントは、Amazon CloudFrontにおける各種制限事項（クォータ）の一覧です。多くの項目はService QuotasやAWSサポートを通じて引き上げが可能です。
- 主要なクォータ項目
- **一般的な制限**
- ディストリビューションあたりの転送レート: 150 Gbps
- URLの最大長: 8,192 バイト
- 1秒あたりのリクエスト数: 250,000
- **ディストリビューション・オリジン**
- アカウントあたりのディストリビューション数: 500
- ディストリビューションあたりのオリジン数: 100
